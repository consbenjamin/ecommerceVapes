# ecommerceVapes
e-commerce de vapes(VapeClub)
- Tecnologias utilizadas:
- Backend: node, express, sequelize, pg, mercadopago, jwt.
- Frontend: react, redux, tailwind.
- DataBase: postgreSQL.
